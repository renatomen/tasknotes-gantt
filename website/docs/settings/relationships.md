# Relationships

*(The whole Relationships group is **companion only** — it's shown only when
TaskNotes is installed. Expansion has no effect in standalone mode, so these
controls are omitted there rather than shown inert.)*

These control how tasks pulled in **for context** — related tasks that fall
outside your Base's filter — are shown. Labels match TaskNotes' own strings.

## Expanded relationships

| Value | Behavior |
| --- | --- |
| **Inherit** *(default)* | Follow TaskNotes' own expansion behavior. |
| **Show all** | Pull in and show out-of-filter related tasks for context. |

## Expanded items opacity (%)

How prominent the pulled-in (out-of-filter) context rows are. Lower = fainter.
**Default:** 55%. **Range:** 10%–100% (they never fully vanish).

Only meaningful with **Expanded relationships → Show all**.

## Hide top-level subtasks

When on, top-level subtasks are hidden from the chart. **Default:** off.
