# Find & focus

Large charts get hard to navigate. **Find & focus** lets you jump straight to any
task.

Open the fuzzy search, type part of a task's name, and pick it. The Gantt then:

- **expands** the task's parents so the row is visible,
- **zooms and scrolls** to bring it into view, and
- **highlights** it briefly.

It's navigation only — focusing a task does **not** open or edit it. The search
includes tasks pulled in through relationships, so you can jump to a related task
even if it isn't a direct row in your Base.
