Where does calendar editing live? — three paradigms, same calendar

Directional wireframes — judge where editing happens, not colours or exact controls. Same data everywhere: calendar “Engineering”, Mon–Fri 09:00–17:00, two holidays, teal colour.

1 · Note opens as the editor (Kanban / Bases pattern)

You click Engineering.md in the file explorer — the note pane itself renders as a visual editor. “Open as markdown” shows the raw YAML any time.

2 · Modal editor (TaskNotes pattern)

The note stays plain markdown. You run a command (or hit “edit” in the calendar picker) — a dialog opens over the workspace; Save writes the YAML back.

3 · Central manager view

A dedicated “Calendars” tab lists every calendar and set with inline editing. You manage everything from one control panel and rarely open the notes.
